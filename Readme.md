# @googlestop_bot — Интеллектуальный конкурс

Telegram-бот для проведения интеллектуального конкурса с главным призом **iPhone 17 PRO 256 Гб**.

## Основные возможности
- 1 бесплатная заявка + платные дополнительные попытки (99 ₽)
- Квиз с таймерами
- Динамическое меню в зависимости от статуса участника
- Прогресс-бар конкурса
- Админ-панель + экспорт данных в Google Sheets
- Полноценный Финал с таймером

## Технологии
- Python 3.12 + aiogram 3.x
- aiosqlite (база данных)
- Railway (хостинг)
- YooKassa (платежи)

## Ветки
- `main` — стабильная версия
- `dev` — разработка новых функций
